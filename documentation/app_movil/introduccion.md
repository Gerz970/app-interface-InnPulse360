# Introducción a InnPulse360

InnPulse360 es una aplicación móvil para la gestión integral de hoteles que permite administrar diferentes aspectos de la operación hotelera de manera eficiente y moderna.

## Funcionalidades principales

- **Autenticación**: Sistema seguro de inicio de sesión
- **Reservaciones**: Crear, ver y gestionar reservas de habitaciones
- **Incidencias**: Reportar y dar seguimiento a problemas en habitaciones
- **Transporte**: Solicitar y gestionar servicios de transporte
- **Limpieza**: Asignar y supervisar tareas de limpieza
- **Clientes**: Gestión de información de clientes (administradores)
- **Hoteles**: Administración de hoteles y sus características (administradores)

## Acceso a la aplicación

La aplicación requiere autenticación mediante usuario y contraseña. Una vez iniciada la sesión, tendrás acceso a los módulos asignados según tu rol.

## Navegación

- Usa el menú lateral (ícono de hamburguesa) para acceder a los diferentes módulos
- Cada módulo tiene su propia pantalla principal con opciones específicas
- El header superior muestra tu información de usuario y opciones adicionales
